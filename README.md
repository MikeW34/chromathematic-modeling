# Хроматематическая модель чисел Фиббоначи и Модуль разности
Описание проекта 
Хроматематическая визуализация — это интерактивное приложение для визуализации математических объектов с использованием хромо-моделей (HMM - хроматические марковские модели). Проект демонстрирует, как различные математические структуры (числа Фибоначчи, 2D функции) могут быть представлены в цвете с помощью различных алгоритмов раскраски.

# Основные возможности
1D визуализация чисел Фибоначчи:
1. Ленточная диаграмма значений по модулю
2. Спираль Фибоначчи с цветовой кодировкой

2D визуализация функций:
1.Тепловая карта для функции Z = |X+Y| - |X-Y|
2. 3D изометрическая проекция

Хромо-модели (4 режима раскраски):
1. Parity — чередование по чётности позиции
2. Value Parity — циклическое повторение каждые 3 элемента
3. Modulo Gradient — циклическое повторение каждые 4 элемента
4. Golden Ratio — паттерн на основе чисел Фибоначчи

Сохранение данных:

Автоматическое сохранение в SQLite базу данных
Возможность заполнения БД тестовыми данными

# Требования для запуска
1. Python 3.8 или выше
2. Использование модулей tkinter и ttk

# Установка
git clone <repository-url>
cd fibonacci-visualization

# Есть возможность установки с помощью виртуального окружения
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate     # Windows

# Запуск
python main.py
# или
python -m gui.main

# Примеры визуализации
Ленточная диаграмма
Цветные столбцы, где каждый цвет соответствует состоянию HMM модели.

Спираль Фибоначчи
Квадраты, вложенные по спирали, с цветовой кодировкой на основе HMM модели.

Тепловая карта
Цветовое представление 2D функции, где цвет показывает значение Z.

3D проекция
Изометрическая проекция с цветовой кодировкой высоты.
